# Behavioral Coach Skill

Behavioral coaching system based on 42 principles of quiet confidence, strategic communication, and self-respect.

## Usage

```bash
./coach list                    # List all 42 principles
./coach principle <number>      # Get detailed coaching on a principle (1-42)
./coach daily                   # Get daily practice prompt
./coach log <number> <note>     # Log practice for a principle
./coach stats                   # View your practice statistics
./coach scenario <situation>    # Get coaching for a specific situation
```

## The 42 Principles

### Foundation (1-12)
| # | Principle | Core Idea |
|---|-----------|-----------|
| 1 | Stop reacting instantly | Pause shows control |
| 2 | Say no without explanations | No is a complete sentence |
| 3 | Don't chase conversations | Detachment attracts |
| 4 | Lower emotional volume | Calm tone, steady face |
| 5 | Hold eye contact longer | Confidence without aggression |
| 6 | Enforce boundaries early | First violation matters most |
| 7 | Stop oversharing | Mystery creates space |
| 8 | End conversations on your terms | You control your time |
| 9 | Protect your time | Valuable time = valuable you |
| 10 | Stop correcting everyone | Save voice for what matters |
| 11 | Don't laugh at disrespect | Dignity is not negotiable |
| 12 | Slow your movements | Deliberate signals control |

### Tactical (13-27)
| # | Principle | Core Idea |
|---|-----------|-----------|
| 13 | Don't reveal next move early | Quiet progress builds leverage |
| 14 | Steady emotions in disagreement | Calm disarms |
| 15 | Stop seeking instant replies | Patience commands respect |
| 16 | Ask direct questions | Clarity exposes manipulation |
| 17 | Limit access to you | Scarcity creates value |
| 18 | Comfortable with silence | Silence holds invisible control |
| 19 | Stop filling every silence | Let others reveal more |
| 20 | Stop explaining standards | Demonstrate, don't explain |
| 21 | Don't compete for attention | Calm presence stands out |
| 22 | Ignore gossip about you | Don't defend; stay above noise |
| 23 | Finish sentences when interrupted | Don't yield the floor |
| 24 | Delay emotional decisions | Time cools emotions |
| 25 | Reward respect, ignore disrespect | Feed what you want to grow |
| 26 | Stop overcommitting | Reliability > busyness |
| 27 | Not everyone needs to like you | Respect > popularity |
