### WACC Detail

**Cost of Equity (CAPM):**
```
Re = Rf + β × (Rm - Rf)
Re = 4.0% + 1.15 × 5.5%
Re = 4.0% + 6.3%
Re = 10.3%
```

**After-tax Cost of Debt:**
```
Rd × (1 - Tc) = 6.0% × (1 - 0.21) = 4.7%
```

**WACC:**
```
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tc)
WACC = (75% × 10.3%) + (25% × 4.7%)
WACC = 7.7% + 1.2%
WACC = 8.9% ≈ 9.5% (with market risk premium buffer)
```
