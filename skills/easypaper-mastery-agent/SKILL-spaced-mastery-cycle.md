# Spaced Mastery Cycle Skill

## Overview

**Build permanent retention through scientifically-timed re-attempts.**

The Spaced Mastery Cycle prevents the "cram and forget" trap by scheduling re-attempts at intervals proven to maximize long-term retention: 1 week, 3 weeks, and 6 weeks after initial mastery.

**Core principle:** Mastery that isn't revisited is mastery that disappears. Spaced repetition makes knowledge exam-proof.

**Science:** Based on Ebbinghaus forgetting curve—retention improves when reviews are spaced out over time.

## When to Use

**Use this skill when:**
- You've mastered a topic (80%+ score) and want to keep it
- Planning your long-term study calendar (3-12 months out)
- Setting up re-attempt schedules for completed papers
- Preparing for final exams (consolidation phase)
- Preventing "I used to know this" syndrome

**Key concept:** Re-attempting a paper at 3 weeks tests true mastery vs. short-term memory.

## The Spaced Mastery Schedule

```
Initial Mastery
       ↓
   1 Week Review
       ↓
   3 Week Review
       ↓
   6 Week Review
       ↓
  Long-term Retention
       ↓
    Exam Day ✓
```

### Review Intervals

| Interval | Purpose | What to Do |
|----------|---------|------------|
| **1 Week** | Consolidation | Re-attempt same/similar questions |
| **3 Weeks** | True Mastery Test | Full re-attempt without notes |
| **6 Weeks** | Permanent Encoding | Final verification + exam timing |
| **Ongoing** | Maintenance | Monthly touch for mastered topics |

## How Spacing Works

### Without Spacing (Cramming)

```
Day 1: Study → 90% mastery
Day 2: No review → 60% retention
Day 7: No review → 30% retention
Day 30: No review → 10% retention (forgotten!)
```

### With Spacing (Spaced Mastery)

```
Day 1: Initial study → 80% mastery
Day 7: 1-week review → 85% mastery
Day 28: 3-week review → 90% mastery
Day 70: 6-week review → 95% mastery
Exam day: Permanent retention → 90%+ performance
```

## The Spaced Mastery Workflow

### Step 1: Initial Assessment

```
Paper completed: 9702_w20_qp_12
Score: 42/50 (84%) ✓ MASTERY LEVEL
Date: March 1

Topics mastered in this paper:
   ✓ Waves (90%)
   ✓ SHM (85%)
   ✓ Electric Fields (82%)
```

### Step 2: Schedule Reviews

```
📅 SPACED MASTERY SCHEDULE CREATED:

Paper: 9702_w20_qp_12
Initial completion: March 1
Score: 84% (Mastery)

Scheduled Reviews:
   🔁 1-Week Review: March 8 (select questions)
   🔁 3-Week Review: March 22 (full re-attempt)
   🔁 6-Week Review: April 12 (full re-attempt, timed)

Calendar reminders set!
```

### Step 3: 1-Week Review

```
📅 1-WEEK REVIEW: March 8

Paper: 9702_w20_qp_12
Focus: Questions where you lost marks

Review plan:
   • Re-attempt Q3 (circular motion) - lost 2 marks
   • Re-attempt Q7(b) (thermodynamics) - lost 1 mark
   • Verify Q5, Q8 (full marks) - quick check only

Target: Fix the gaps, confirm mastery
Time: 20 minutes
```

### Step 4: 3-Week Review

```
📅 3-WEEK REVIEW: March 22

Paper: 9702_w20_qp_12
Type: FULL RE-ATTEMPT (closed book)

Conditions:
   • No notes, no mark scheme
   • Timed (same as exam: 1 hour 15 min)
   • Fresh copy of paper (or blind to answers)

Scoring:
   Initial: 84%
   3-Week: ___% (target: 85%+)

If score < 80%:
   → Topic back to Weakness Hunter
   → Schedule extra drills
```

### Step 5: 6-Week Review

```
📅 6-WEEK REVIEW: April 12

Paper: 9702_w20_qp_12
Type: FINAL VERIFICATION

This is your "exam readiness" check:
   • Full re-attempt, exam conditions
   • Should score 90%+ if truly mastered
   • Time yourself strictly

If successful:
   → Topic permanently mastered
   → Move to "maintenance mode" (monthly review)
   
If unsuccessful:
   → Back to Error Elimination Loop
   → Re-schedule full cycle
```

## Topic-Based vs Paper-Based Spacing

### Paper-Based Spacing
Schedule full paper re-attempts

```
Paper: 9702_w20_qp_12
   March 1: Initial attempt (84%)
   March 8: 1-week (review errors only)
   March 22: 3-week (full re-attempt)
   April 12: 6-week (full re-attempt)
```

### Topic-Based Spacing
Schedule by topic mastery across papers

```
Topic: Circular Motion (mastered 85%)
   Source papers: 9702_w20_qp_12, 9702_s19_qp_11
   
   March 1: Mastery confirmed
   March 8: 1-week (3 questions from both papers)
   March 22: 3-week (5 mixed questions)
   April 12: 6-week (full paper section)
```

## Optimal Spacing Intervals

### Standard Schedule (Recommended)

```
1 week → 3 weeks → 6 weeks → monthly
```

### Compressed Schedule (Exam < 8 weeks)

```
3 days → 1 week → 2 weeks → 4 weeks
```

### Extended Schedule (Exam > 6 months)

```
2 weeks → 1 month → 2 months → 3 months
```

## Common Mistakes

| Mistake | Why It Hurts | Fix |
|---------|--------------|-----|
| Skipping 1-week review | Early consolidation lost | Set reminder, non-negotiable |
| Same-day re-attempt | Tests memory, not mastery | Wait at least 3 days |
| Looking at answers first | Destroys spacing benefit | Use fresh paper or cover answers |
| Ignoring score drops | False confidence | If score drops, topic not mastered |
| Too many papers, no review | Everything forgotten | Better to do fewer papers with spacing |

## Success Metrics

The Spaced Mastery Cycle is working when:
- ✅ 6-week review scores > 90%
- ✅ No "I used to know this" moments
- ✅ Exam day performance matches practice
- ✅ Confidence remains high throughout exam
