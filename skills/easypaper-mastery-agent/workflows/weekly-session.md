# Weekly Deep Session Workflow

**Duration:** 60-90 minutes  
**Frequency:** Every Sunday (or chosen day)  
**Trigger:** Student says "Weekly review"

## Purpose

The weekly deep session consolidates learning, eliminates accumulated errors, and plans the week ahead. It's where the system "catches its breath" and ensures nothing falls through the cracks.

## Session Structure

```
Total Time: 60-90 minutes

Phase 1: Error Elimination (20 min)
Phase 2: Mini Weakness Hunter (30 min)
Phase 3: Next Week Scheduling (10 min)
Buffer: Wrap-up and questions (10-30 min)
```

## Post-Session Actions

1. **Download drill questions** to specified folders
2. **Set calendar reminders** for spaced reviews
3. **Prepare papers** for next week's new attempts
4. **Update tracking sheets** with new data
