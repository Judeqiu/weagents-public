## Error Categories

Use these codes in the "Error Type" column:

- **C** = Conceptual (wrong understanding)
- **CAL** = Calculation (arithmetic/algebra)
- **CAR** = Careless (rushing, misreading)
- **T** = Technique (exam presentation)
