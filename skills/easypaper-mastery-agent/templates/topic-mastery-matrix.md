## How to Update This Matrix

### After Each Paper:
1. Add question attempts to relevant topic rows
2. Update "Qs Attempted" and "Qs Correct"
3. Recalculate Mastery % = (Correct / Attempted) × 100

### Monthly (Weakness Hunter Run):
1. Update all mastery percentages
2. Re-assign status colors (🟢🟡🔴)
3. Update "Monthly Progress Tracking" table
4. Refresh "Priority Actions" list
5. Set new targets and deadlines

### Target Benchmarks:
- **3 months before exam:** All topics at least 🟡 Developing (50%+)
- **1 month before exam:** All topics at least 🟢 Mastery (80%+)
- **Exam week:** Focus on 🟢 topics only (confidence building)
